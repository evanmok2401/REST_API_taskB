# REST_API_taskB

A simple Javascript Backend and REST API to GET, POST, PUT, DELETE. Tests are written using Mocha and Chai. This repository serves as a self-learning exercise.

# Instructions
To run locally:
```
npm install
npm start
```

To run tests:
```
npm install
npm test
```
